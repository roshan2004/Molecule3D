# Roadmap

Potential next steps:

- Robust mmCIF support through a dedicated parser or optional dependency.
- More descriptor families and stable preset feature sets.
- Notebook examples for common teaching and ML workflows.
- More explicit coarse-grained topology objects.
- Export formats for coarse-grained topology prototypes.
- Optional API documentation generation from docstrings.
- More graph featurization presets for ML.
- Documentation deployment through GitHub Pages.
